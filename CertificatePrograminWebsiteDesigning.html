About Course

In today’s scenario, an organization makes a global presence through the internet. This requires designing of websites. The designing of websites involves creation. Therefore, it is essential for designers to created appealing. Aesthetic and interactive web pages. Web pages created in HTML are static in nature. They do not interact with the user and therefore, lack appeal. To maintain a continual interest of the user in the web page and the website, the web pages must be dynamic. This dynamism can be added in the form of animations, message pop ups and flashing images. Scripting languages are used to add dynamism to static HTML web pages. JavaScript is a popular scripting language supported across web browsers.
